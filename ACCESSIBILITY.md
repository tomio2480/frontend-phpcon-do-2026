## 実装済みのアクセシビリティ対応（Phase 11）

### 地図コンポーネント（`HokkaidoMap`）

- 地図コンテナに `role="application"` と `aria-label="北海道地図"` を付与
- SVG ポリゴンに `role="button"` / `tabindex="0"` を付与（Tab 順序に組み込み）
- SVG ポリゴンに `aria-label`（市区町村名）と `aria-pressed`（選択状態）を付与
- Enter キーおよび Space キーでポリゴンを選択・解除できる
- フォーカス時にホバースタイルを適用し，スクリーンリーダーと視覚的 UI が連動

### チェックボックスリスト（`CheckboxList`）

- `<label>` でチェックボックスを囲む実装により，クリック範囲とラベル紐付けを確保
- Tab キーで各チェックボックスにフォーカスを移動できる

### レスポンシブ対応

- モバイル幅（375px）では結果パネルが地図より上部に表示される
- 結果パネルはモバイル時に `sticky top-0` で常時表示される
